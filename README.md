# DATA 205 Final Project

### Overview

In this project I am analyzing the effiency of neighborhood bus routes in Montgomery County Maryland. The goal is to determine neighborhoods that lack proper access to transit stations, and identify areas of need among these neighborhoods by comparing Census estimates on population, household size and number of vehicles to accessibility to nearby bus routes.

### Tools Used

* R (RStudio & Quarto)
* ArcGIS
* PowerBI

### Project Structure

This is the workflow structure for this repository:

* data/ raw and processed datasets
* ingestion/ data loading and cleaning
* eda/ exploratory data analysos
* analysis/ modeling and results
* reports/ final outputs and presentations
